# AI Chat Client

A vibe-coded chat client for a local LLM hosted at `http://0.0.0.0:8000/v1`.

## Features

- Modern, aesthetic UI
- Real-time chat interface
- Connects to local OpenAI-compatible API

## Setup

npm install
npm run dev --host

# Kill server 
pkill -f "npm run dev" || true

# Start server
